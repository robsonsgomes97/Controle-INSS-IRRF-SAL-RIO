Cálculo de INSS, IRRF e Salário Líquido no Excel

Este projeto consiste em uma planilha desenvolvida no Excel para simular o cálculo de folha de pagamento, incluindo os principais descontos obrigatórios no Brasil: INSS e IRRF.

🚀 Funcionalidades
Cálculo automático do INSS com base no salário bruto
Cálculo do IRRF considerando as faixas salariais
Cálculo do Salário Líquido (Bruto - INSS - IRRF)
Classificação visual por faixa salarial (cores e indicadores)
Criação de ranking de salários líquidos utilizando Tabela Dinâmica
Uso de formatação condicional com ícones para análise visual
Estrutura organizada para facilitar análises financeiras
🧮 Fórmulas Utilizadas
SE (condicional)
SUBTOTAL
Cálculos percentuais
Referências absolutas e relativas
Formatação personalizada (R$)
📈 Recursos do Excel Aplicados
Tabela Dinâmica
Segmentação de Dados
Formatação Condicional (cores e ícones)
Layout profissional para análise de dados
<hr>
🎯 Objetivo

Praticar e demonstrar conhecimentos em Excel voltados para rotinas administrativas e financeiras, como controle de folha de pagamento e análise de dados.
<hr>
📌 Observação

Os cálculos de INSS e IRRF foram aplicados de forma simplificada para fins didáticos.
